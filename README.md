# Intel 8080 Emulator (for Space Invaders)

An Intel 8080 emulator using SDL, meant for running Space Invaders by Taito

# Input

| Key     	| Description              	|
|---------	|--------------------------	|
|    C    	| Insert coin              	|
| [Space] 	| P1 Start                 	|
|   A/D   	| P1 move Left/Right       	|
|    W    	| P1 shoot                 	|
|    T    	| Tilt machine (Game Over) 	|
